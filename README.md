# Carmen Academic Site

Static academic website for GitHub Pages.

## Files to edit

- `index.html`: update paper titles, abstracts, teaching entries, and any profile text.
- `styles.css`: visual styling.
- `assets/docs/Carmen_Armas_Montalvo_CV.pdf`: replace with your CV PDF.
- `assets/docs/papers/`: add paper drafts here if you want public draft links.
- `assets/docs/slides/`: add presentation slides here if you want public slide links.

## Recommended content choices

- Research: yes, include at least a short abstract for each paper.
- Drafts/slides: optional. Add them only for papers you want to share publicly.
- CV: yes, this section can be just one uploaded PDF.

## Deploy on GitHub Pages

1. Create a GitHub repository.
2. Upload the contents of this folder.
3. In GitHub, go to `Settings > Pages`.
4. Set the source to deploy from the main branch root.
